Resizes a string.

Parameters
[in,out]sAn initialized string.
[in]nThe new size.
Warning
If the string size increases, the new characters will not be initialized. This will leave the end of s with garbage. If this is inconvenient for your usage, use rs_resize_w().
Allocation
When n is greater than the capacity of s.

Constant.
